# RAG для службы поддержки

Система автоматизации ответов службы поддержки на основе RAG (Retrieval-Augmented Generation). В проекте в качестве примера используется сгенерированный файл с информацией о несуществующем банке. 

## Ключевые особенности

*   Hybrid Search: комбинация семантического поиска (векторное сходство) и ключевого поиска (BM25) для учета как смысла вопроса, так и точных совпадений терминов.
*   Query Rewriting: переформулировка запросов позволяет корректно обрабатывать уточняющие вопросы пользователей, восстанавливая пропущенные сущности.
*   Reranking: этап переранжирования найденных чанков для отбора наиболее релевантного контекста перед генерацией ответа.

## Стек

*   HF Transformers, PyTorch, LangChain, FAISS, Gradio
